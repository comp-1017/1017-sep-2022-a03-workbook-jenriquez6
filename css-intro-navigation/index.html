Create navigational links within a website

There are several techniques to navigate within a web site. The semantic tag for navigation is the <nav> tag. You can navigate through your website by using relative addressing. For our examples, we will use the unorder list tag (<ul>).

  • page to page within the same folder
  • page to page between folders
  • staying on the current page
  • going to a specific place on the current page (on page navigation)

Navigational menus are placed in your <header> semantic area. They may also be placed in your <footer> area as a sitemap. The nav tag like other semantic tags should a <hx> tag. However this header is optional. If you omit the header tag, you will receive a document outliner error: Untitled section. Until you are shown how to avoid this error message, it will be acceptable on your document outline.

CSS Coding Styles Summary

Browser Defaults you have no control over the styling
Inline this is for a single specific element (tag) instance. your css will be specified in a parameter within the element. This css affects only the element tag that contains the specification. style="ryle; [rules; ...]" style = "color:red;"
Embedded the specification of css as embedded happens in your <head> area. are this css affects all similar elements on the web page where the css is coded. <style> selector{rule; [rule;.....]} [selector {rule; [rule;...]}...]</style> <style> p{color: red;width:50px} h1 {font=style:italic}</style>
link (stylesheet) preferred method of doing css. you place your css in a separate physical file. You place your css in a separate physical file. This file can then be used on any web page in your web site . The link to this file will be done in your <head> area. The css in this file is similar to embedded EXCEPT there is no style tag selector{rule; [rule;.....]} [selector {rule; [rule;...]}...] p{color: red;width:50px} h1 {font=style:italic}
